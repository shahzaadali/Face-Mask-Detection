😷 Face Mask Detection using CNN

📌 Project Overview

This project focuses on detecting whether a person is wearing a face mask or not using a Convolutional Neural Network (CNN). The model is designed to work on both images and real-time video streams, making it useful for public safety monitoring systems.

🚀 Features

Detects mask and no-mask faces in images and video

Real-time monitoring using webcam/video stream

High-performance CNN model built with deep learning techniques

Visual representation of predictions and performance metrics

🧠 Technologies Used

Python

TensorFlow

Keras

OpenCV

NumPy

Matplotlib

Seaborn

📂 Dataset

Collected a balanced dataset containing images of:

People wearing masks

People not wearing masks

Applied preprocessing techniques such as:

Image resizing

Normalization

Data augmentation (if applicable)

⚙️ Model Architecture


Convolutional Neural Network (CNN)

Multiple convolution + pooling layers

Fully connected (Dense) layers

Activation functions: ReLU & Softmax

📊 Model Evaluation

The model was evaluated using the following metrics:

Accuracy

Precision

Recall

Confusion Matrix

📈 Results Visualization

Performance graphs plotted using Matplotlib and Seaborn

Confusion matrix visualization for better understanding of predictions
